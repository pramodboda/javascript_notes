## Data types
The latest ECMAScript standard defines seven data types:
- Six data types that are primitives:
  - Number
  - String
  - Boolean
  - Undefined
  - Null
  - Symbol (new in ECMAScript 2015) 
- and Object

```javascript
typeof 143
=> "number"
```
```javascript
typeof "Pramod"
=> "string"
```
```javascript
typeof true
=> "boolean"
```
```javascript
typeof undefined
=> "undefined"
```
```javascript
typeof null
=> "object"
```
```javascript
typeof []
=> "object"
```
```javascript
typeof {}
=> "object"
```
```javascript
typeof NaN
=> "number"
```
